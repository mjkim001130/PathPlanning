CROMA: Companion Robot with Object Manipulation and Autonomy
Currently, the robot arm has been programmed to pick up objects using PRM-based path planning. More features are being developed and will continue to be updated.

Abstract

This project aims to develop a companion robot for home environments, allowing users to control the robot via voice or text commands to perform tasks. The robot responds to user commands by executing actions such as picking up or moving objects (pick-and-place), making it a practical tool for household assistance.

For this project, CoppeliaSim is used as the simulation environment, and the KUKA YouBot model is employed to simulate and test various tasks. To ensure accurate pathfinding and object manipulation, path planning techniques such as Probabilistic Roadmap (PRM) and Rapidly-exploring Random Tree (RRT) are integrated, enabling the robot to navigate around obstacles while reaching target positions.

The anticipated impact of this project is to enable the robot to serve as a practical household companion, providing daily assistance with object handling and task support. This functionality has potential applications as an assistive tool for the elderly or disabled, ultimately improving household efficiency and enhancing the quality of life.

Video

PRM

The Probabilistic Roadmap (PRM) enables the robot arm to navigate around obstacles to reach and pick up objects. By generating a roadmap of collision-free paths, PRM helps the robot plan an efficient route to its target.

RRT

The Rapidly-exploring Random Tree (RRT) algorithm finds a path by growing a tree from the start point toward random samples, connecting each step closer to the target while avoiding obstacles.
